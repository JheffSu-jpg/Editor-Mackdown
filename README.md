# 📝 Editor de Markdown

Um editor de Markdown moderno com preview ao vivo, desenvolvido com HTML, CSS e JavaScript puro.
O projeto foi criado com foco em aprendizado, prática de manipulação de DOM e construção de um projeto real para portfólio front-end.

---

## 🚀 Funcionalidades

✅ Preview em tempo real
✅ Conversão de Markdown para HTML
✅ Tema dark/light
✅ Salvamento automático com localStorage
✅ Contador de palavras e caracteres
✅ Botões de formatação rápida
✅ Exportar arquivo `.md`
✅ Copiar Markdown com um clique
✅ Responsivo para desktop e mobile

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
